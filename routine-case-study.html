Child's Summer Routine Design System

Solo UI/UX & Asset Production Designer | GIT Case Study Portfolio Piece

The Problem

Children navigating Levels 1 and 2 Autism often experience heightened transition anxiety when daily expectations are abstract, overwhelming, or unpredictable. Traditional visual schedules frequently fail because they pull from mismatched, visually noisy stock illustration packs.

This structural and stylistic inconsistency forces a neurodivergent child to constantly decode entirely new visual rules for every single task. The resulting cognitive friction increases sensory overload, undermines routine comprehension, and hinders successful behavioral transitions.

The Solution

The Child's Summer Routine Design System resolves this friction by establishing a highly structured, single-brand design ecosystem. By programmatically stripping background noise and engineering a unified Character DNA across all assets, the design system drastically minimizes cognitive load.

It scales seamlessly across linear daily schedules, milestone tracking games, and loose token economies. The final physical-digital system utilizes a rigid horizontal layout grid and a calming, circadian color taxonomy to give the child a highly accessible, predictable roadmap for their day.

Technical Execution & UI Metrics

  • Tri-Band Color Taxonomy: Applied a contextual color shift to regulate circadian expectations: Teal-Cyan for Morning routines, Fresh Green for Mid-Day activities, and Cobalt Blue for Evening/Bedtime assets.
  • Rigid Grid System: Engineered a universal horizontal grid framework across all core columns. Every routine card shares a standard dimensional layout and white-box padding to isolate the focus task.
  • Typographic Hierarchy: Consolidated the entire project into a maximum of two sans-serif typefaces. Implemented strict Title Case styling and wide-tracked, uppercase header formatting to establish alignment and modern, clean text blocks.

Technical Asset Optimization

  • Stylistic Harmonization: Programmatically deconstructed multiple mismatched Adobe Stock vector files to isolate primary subjects. Background clutter was entirely stripped to eliminate sensory distractions.
  • Character DNA Engineering: Extracted a single child asset to build a master character template layer. Using a precise mechanical workflow, the child's face and distinct eye shapes were extracted, duplicated, and scaled across different bodies and family vignette graphics to create total visual continuity.
  • Advanced Illustrator Workflows: Leveraged asset-building tools, scaling, object transformation axes, and duplicating functions to cleanly blend anchor paths at complex limb and neck intersections. Thickened thin outline weights to enforce a single line-art standard across all artboards.
  • Dynamic Background Composition: Executed an intentional full-bleed pattern layer on the Bedtime Pass asset, utilizing a vector clipping mask to cleanly crop runaway shapes at the artboard boundaries for a sharp, production-ready edge.

Scalability & Token Economies

  • The Potty Path Loop: Scaled the core components into a linear milestone tracking game. Re-engineered the path using a fluid S-curve vector flow, uniform connecting lines, and clean number graphics to keep the tracking experience highly accessible.
  • Constellation Tracking & Interaction: Designed an open, scatter-style token economy sheet (Stay in Bed) alongside interactive, printable cutout token sheets (Potty Tokens and Star Tokens), maintaining identical grid math, outline weights, and object scales.
